# 🎬 NineCast — Projeto de Português

> Um site criado com o objetivo de apresentar conteúdos de forma simples, criativa e interativa para um trabalho escolar de Português.

🔗 **Acesse o projeto online:**  
👉 https://ninecast.netlify.app/

📂 **Repositório no GitHub:**  
👉 https://github.com/henriq-ycler-js/NineCast-projeto-de-portigues

---

## 📌 Sobre o projeto

O **NineCast** é um site desenvolvido como atividade escolar, com foco em apresentar conteúdos de forma digital, substituindo métodos tradicionais por uma experiência mais moderna.

O projeto foi construído utilizando tecnologias básicas da web, com o objetivo de demonstrar organização, criatividade e domínio inicial de desenvolvimento front-end.

---

## 🚀 Funcionalidades

- 📄 Exibição de conteúdo textual  
- 🎨 Interface estilizada com CSS  
- 🌐 Hospedagem online (Netlify)  
- 📱 Layout simples e acessível  
- ⚡ Carregamento rápido (site leve)

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (básico)  
- Git & GitHub  
- Netlify (deploy)
